# 规则漏斗

## 最小输入

输入为有确定顺序的规则配置、分析单位、规则字段和确认的拒绝码策略。必须确认漏斗按规则顺序还是按已有拒绝码顺序重建；独立命中统计不得替代漏斗结果。按需读取指标词典第 2、5、6 节。

## 计算顺序

1. 从确认的完整分析总体开始，记录第 1 层进入样本。
2. 每层仅在上一层剩余样本中执行当前规则，记录新增命中和剩余样本。
3. 计算累计拒绝、累计拒绝率及有成熟标签时的层级风险表现。
4. 将下一层进入量与上一层剩余量逐层核对；不允许跳过或重置样本总体。

## 输出与披露

输出 `rule_funnel_summary.csv`，至少含层级、规则代码、进入量、新增命中、剩余量、累计拒绝、比例和可观测风险。报告说明规则版本、顺序、分析颗粒度、成熟过滤；拒绝无表现指标标为不可观测。

## 验收

下一层进入量等于上一层剩余量，`进入量 - 新增命中 = 剩余量`，累计拒绝量单调不减。前后颗粒度变化时先确认聚合并重新计算，不得拼接两个漏斗。
